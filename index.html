使用须知

  • 我们提供的服务仅仅是：依据您所查询的条件在互联网中收集到的数据中检索相关内容。
  • 我们不提供：文件的上传、保存，链接的检测与验证。
  • 请注意，所有链接均为程序自动抓取自互联网公开资源，我们对链接内容、安全性、有效性一无所知。请下载时务必验证下载内容与检索信息一致，并使用杀毒软件或沙盒进行处理。

侵权内容投诉

删除索引： 如果您发现本网站的搜索结果中错误地展示了您具有版权的信息，请联系我们 zhelper@proton.me ，我们会第一时间将相应结果从网站数据库中移除。
删除内容： 但由于我们只是通过数据收集等方式从互联网公开资源中获取了一些元数据，即使在本网站删除了索引，相应的侵权文件仍然存在于互联网的某一网站。请参照以下内容联系文件传播者删除文件：
  • 标为 z-library 的数据: 这些数据来自 z-library 和 annas-archive 以及 Github 开源项目 zlibsearch，三者网址分别为：https://singlelogin.me ，https://annas-archive.org/ ，https://github.com/zlib-searcher/zlib-searcher。zhelper v5 网站所使用的相关 cid、md5、文件大小信息从后两者获取。zhelper v4 使用的接口通过对前者网站上下载的 app 进行分析后提取（仅供学习交流使用！）。其中，所谓秒传码均为依据数据库中MD5值、文件大小进行推测得到，也未对文件情况进行任何验证。
  • 标为 sslibrary 的数据：这些数据来自于网上流传的所谓秒传书表，本网站所使用的版本来自于 https://forum.freemdict.com/ （相关帖子已删除，我们可以协助提供文本文档，名为“500w电子书.txt”）。请注意，所谓秒传码均为依据数据库中MD5值、文件大小进行推测得到，也未对文件情况进行任何验证。
  • 标为 sanqiu 的数据：链接爬取自三秋书屋： https://www.sanqiu.mobi/
  • 标为 fast8 的数据：链接爬取自fast8： http://fast8.cc/
  • 标为 shuzhou 的数据：链接爬取自书舟： https://www.shuzhou.cc/
